#CSSLint

Validate your CSS code with CSSLint.

##Installation

- Download the zip file and unzip it to your plugin folder.

##Requirements
- Requires modern browser with web worker.
  - FF >= 3.5
  - Chrome >= 4
  - IE >= 10
  - Opera >= 10.6
  - Safari >= 4
  - Details: [caniuse.com](http://caniuse.com/#feat=webworkers)
